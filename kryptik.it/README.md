# kryptik.it

Public website for Kryptik Research Group.

The site is a static SvelteKit project that introduces the lab, documents current research themes, and links the surrounding project network.

## Development

```sh
npm install
npm run dev
```

## Validation

```sh
npm run check
npm run build
```

The app is prerendered with `@sveltejs/adapter-static`; production output is written to `build/`.
